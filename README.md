# RadiantAI: Clinical Governance Platform

A high-fidelity medical diagnostic system integrating AI analysis with the Sentinel Protocol.

## Features
- **Pneumonia & Fracture Detection**: Automated radiology analysis.
- **Sentinel Protocol**: Immutable hash-chained audit ledger.
- **Privacy Firewall**: Multi-doctor identity system with assigned-care logic.
- **E-Pharmacy**: Secure prescription marketplace.
